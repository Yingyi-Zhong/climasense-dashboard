# ClimaSense Dashboard
ClimaSense is an interactive weather insight platform designed to help Los Angeles residents make **data-informed daily decisions**.  

By integrating real-time weather and air quality data, it offers **personalized recommendations** for outdoor activities, comfort levels, and clothing choices, helping users adapt to changing environmental conditions.


## Live Demo
[View on GitHub Pages](https://yingyi-zhong.github.io/climasense-dashboard/)


## Features
- **Real-Time Weather Insights:** Displays up-to-date temperature, humidity, and wind conditions.  
- **Air Quality Visualization:** Integrates Open-Meteo Air Quality API for real-time AQI data.  
- **Personalized Suggestions:** Recommends suitable activities and outfits based on weather comfort.  
- **Interactive Dashboard:** Clean and responsive layout optimized for both desktop and mobile.  
- **Feedback Form:** Allows users to send suggestions or questions directly through the contact form.


## Data Sources
This dashboard uses **free, open-source APIs** to ensure accurate and up-to-date information:
- [Open-Meteo Weather Forecast API](https://open-meteo.com/en/docs)
- [Open-Meteo Air Quality API](https://open-meteo.com/en/docs/air-quality-api)
